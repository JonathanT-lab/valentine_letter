# 💖 Valentine Interactive Web Project

This project is a fun, interactive Valentine-themed webpage built with HTML, CSS, and JavaScript.

## 🌐 Live Demo
https://JonathanT-lab.github.io/valentine_letter

## ✨ Features
- Interactive YES / NO buttons
- Moving "NO" button effect 🎯
- Redirect to a letter page
- Clean and aesthetic design
- Image display support

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📖 Project Overview
This project demonstrates how a simple webpage can be enhanced with interactivity and user engagement using JavaScript.

It simulates a playful Valentine interaction, making the experience more dynamic than a static page.

## 🚀 How It Works
1. The user is presented with a Valentine's question
2. Hovering over "NO" moves the button randomly
3. Clicking "YES" shows a message
4. Page redirects to a letter page
